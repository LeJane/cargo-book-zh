# The Cargo Book

![Cargo Logo](images/Cargo-Logo-Small.png)

Cargo 是[Rust]的 _包经理_。Cargo 会下载您 Rust 的包依赖项,编译您的包,生成可分发的包,并将它们上传到[crates.io] - Rust 社区的*包注册表*。你可以为这本书做出贡献在[GitHub].
